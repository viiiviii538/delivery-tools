# 📱 配信記録PWAアプリ - セットアップガイド

## 🎉 完成した機能

このPWA（Progressive Web App）は、iPhoneやAndroidのホーム画面に追加して、ネイティブアプリのように使用できます。

### ✨ 主な機能

#### 1. **ホーム画面**
- 今月の総売上、平均時給、セッション数、平均客数を表示
- 売上内訳を円グラフで可視化（入場料、チップ、特別報酬）
- ワンタップで新規記録追加

#### 2. **記録画面**
- スマホ最適化された入力フォーム
- 星評価（★）で体調・モチベーションを直感的に入力
- 日付・時刻は自動入力（編集可能）
- 自動計算項目：稼働時間、時給、各種率、曜日、時間帯
- データはLocalStorageに保存（オフライン対応）

#### 3. **カレンダー画面**
- 2026年2月のカレンダー表示
- 配信実施日は紫色でハイライト表示
- 日付タップで詳細データ表示
  - 総売上、時給、稼働時間
  - 体調・モチベーション（星表示）
  - イベント有無
  - 成功メモ

#### 4. **統計画面**
- **売上内訳円グラフ**: 入場料・チップ・特別報酬の比率
- **日別売上推移グラフ**: 折れ線グラフで売上変化を可視化
- **客ファネル円グラフ**: 総客数→コインあり→有料移行の転換率
- **曜日別平均時給**: 棒グラフで最適な配信曜日を特定

## 📲 iPhoneでの使い方

### ステップ1: ファイルをWebサーバーにアップロード
1. `streaming-record-pwa.zip` を解凍
2. 以下のファイルをWebサーバーのルートディレクトリにアップロード:
   - index.html
   - app.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png

### ステップ2: iPhoneでアクセス
1. Safariで `https://あなたのドメイン.com/index.html` を開く
2. 画面下部の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. アプリ名を確認して「追加」をタップ

### ステップ3: アプリとして起動
- ホーム画面のアイコンをタップするだけでアプリが起動
- ブラウザのアドレスバーが表示されず、ネイティブアプリのような体験

## 📊 データ構造

### LocalStorageに保存されるデータ
- **キー**: `streamingRecords`
- **形式**: JSON配列
- **内容**: 
  - 手入力データ: 日付、時刻、デバイス、体調、客数、売上など
  - 自動計算データ: 稼働時間、時給、各種率、曜日、時間帯

### データのバックアップ
ブラウザのLocalStorageに保存されるため、以下の点に注意:
- ブラウザのキャッシュクリアでデータ消失の可能性
- 定期的に開発者ツールからJSONエクスポートを推奨
- 将来的にはクラウド同期機能を追加可能

## 🎨 カスタマイズ

### 色テーマの変更
`index.html` の `<style>` セクションで以下を編集:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### 項目の追加
`app.js` の `saveRecord()` 関数にフィールドを追加:
```javascript
const newField = document.getElementById('newField').value;
```

### グラフのカスタマイズ
Chart.jsのオプションを編集してグラフの見た目を変更可能

## 🐛 トラブルシューティング

### Q: データが保存されない
A: ブラウザのCookieとサイトデータを許可してください

### Q: グラフが表示されない
A: インターネット接続を確認してください（Chart.jsをCDNから読み込み）

### Q: iPhoneのホーム画面に追加できない
A: 必ずSafariで開いてください（Chrome不可）
